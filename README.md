一、需求
###1. 项目概述

####1.1 项目名称
	英文：Egoless do
	中文：心流纪
####1.2 Slogan（口号建议）
	英文：Awaken Your Inner Universe
	中文：唤醒内在宇宙，每一天都是新开始

####1.3 产品定位
	Egoless Do (心流纪) 是一款跨平台间歇性禁食 + 正念修行 + 运动习惯 + 思维记录的综合身心成长应用。它帮助用户养成健康饮食、规律冥想/修行、户外运动习惯，同时随时捕捉内心感悟与灵感，实现身、心、思全面提升。融合科学数据追踪、正念仪式感、社区连接与个人思维管理，提供完整闭环体验。让用户在行动中觉察、在记录中成长、在连接中坚持，实现身（习惯）、心（正念）、思（感念）的全面提升。

	核心理念：把每一次打卡、每一次感悟，都变成宇宙中属于自己的小小星辰。

####1.4 目标用户

	追求身心平衡、习惯养成、自我成长的用户。
	希望记录生活感悟、追踪思维演进、获得长期动力的用户。

####1.5 核心价值主张
	科学 + 仪式感 + 连接 + 自我觉察：数据追踪 + 正念体验 + 社区归属 + 思维记录与成长可视化。
	高留存设计：宽限期 + 多维度激励 + 地图社区 + 个人感念时间轴。
	隐私优先：核心数据本地化，可选匿名社区参与。

###2. 核心功能需求（精炼优化版）

####2.1 每日核心打卡（MVP 核心）
	支持自定义“禁食截止时间”。
	智能宽限期（0-2 天）。
	打卡时支持快捷感悟评论（限制 20 个字）。
	每日智能提醒。

####2.2 禁食会话
	自定义时长、实时倒计时、自动估算热量。
	结束总结 + 感悟评论（≤20 字）。

####2.3 正念修行模块
	冥想/打坐（预设时长、环境音、呼吸引导、结束铃声）。
	其他快捷修行 + 感悟评论。
	一键分享成就卡片。

####2.4 运动记录模块（参考 Keep）
	快走、慢跑、散步、自由跑步。
	GPS 轨迹、语音播报、实时地图、结束总结 + 感悟评论（≤20 字）。

####2.5 健康数据记录
	饮食、体重、饮水等记录 + 感悟评论（≤20 字）。

####2.6 数据统计与激励系统
	多维度连胜、正念点数、成就系统。
	个人打卡时间轴：每日节点展示完成的习惯及对应感悟评论。

####2.7 感念模块（Mind Reflections / Insight Cards）【新增】

	模块目标：让用户随时随地记录闪现的想法、感悟、灵感或内心独白，形成个人思维成长轨迹，提升自我觉察能力。

	核心功能：随时记录：支持首页悬浮按钮、快捷入口或 Widget（小组件）快速创建感念。

	Mind 卡片生成：
	自动生成精美卡片（包含日期、时间、内容、背景图或渐变色）。
	支持手动编辑内容（文字为主，支持少量表情）。

	标签系统：为卡片添加一个或多个标签（预设 + 自定义），如：#感恩、#觉察、#压力释放、#创意灵感、#健康感悟、#跑步心得 等。

	感念历史时间轴：
	以时间倒序或时间轴形式展示所有 Mind 卡片。
	支持按日期浏览、按标签筛选（单标签或多标签组合）。
	筛选后动态生成专注时间轴，帮助用户追踪某一类想法的演进过程（例如“查看所有#压力释放标签的记录，看见自己的心路历程”）。
	支持卡片编辑、删除、置顶、分享单张 Mind 卡片。

	统计洞察：
	感念总数、最高频标签、连续记录天数。
	可视化：标签云、每月感念数量趋势。
	与其它模块联动：
	打卡/运动/修行结束时可一键转为感念卡片。
	感念可关联具体习惯记录。



####2.8 社区与社交功能
	匿名排行榜、打卡圈。
	共享习惯打卡地图：实时显示全球同频者，支持筛选习惯，查看脱敏感悟（≤20 字）。

####2.9 设置与个性化
	主题切换 + 订阅高级主题 / 音效包（含 Mind 卡片高级模板）。
	标签管理（自定义默认标签）。
	位置共享、通知、隐私设置。
	首次启动引导页 + 设置向导。

###3. 非功能需求

####3.1 性能与体验

	感念记录支持极低延迟（离线即时保存）。
	时间轴与标签筛选流畅，支持大量数据（推荐本地数据库）。
	Mind 卡片支持优雅动画与分享图片生成。

####3.2 隐私与数据安全
	感念内容完全本地存储，默认不上传社区。
	用户可手动选择将单张 Mind 卡片匿名发布到打卡圈或地图。

####3.3 商业化

	Premium 订阅：高级 Mind 卡片模板、更多标签颜色、无限历史搜索、智能标签建议、导出 PDF 时间轴等。

####3.4 其他需求

	跨平台：iOS + Android + web 良好体验（含 Samsung 底部栏适配）。
	本地化：强响应式设计（responsive.ts 处理不同设备尺寸，包括 iPad）。
	性能：离线优先（AsyncStorage 持久化），动画流畅（Reanimated）。
	隐私：本地存储为主，有 PRIVACY_POLICY.md，支持健康数据同步。
	国际化：首次启动自动检测设备语言，完整翻译。
	可访问性/UI：卡片式设计、分享功能、启动页/欢迎页。
	构建与发布：Expo EAS Build，支持 Android R8 混淆减小 APK 体积。


###4. 核心数据模型（更新）

	HabitRecord：习惯打卡记录（含 ≤20 字感悟）。
	MindReflection（新实体）：
	id, timestamp, content（文字），tags（字符串数组），mood（可选），linkedHabitId（可选关联）。
	TimelineEntry：统一用于打卡时间轴与感念时间轴。

###5. 用户流程亮点

	日常路径：快速打卡/运动 → 记录感悟 → 随时新建 Mind 卡片 → 时间轴回顾成长轨迹 → 地图感受全球连接。
	感念专属路径：灵感闪现 → 打开 App 快速记录 → 添加标签 → 后期通过标签筛选追踪同一主题的思考演变。
	激励循环：完成习惯 → 正反馈 → Mind 卡片沉淀思考 → 社区/地图分享 → 持续成长可视化。


###6. 未来扩展方向（Roadmap）

	Phase 1：MVP（打卡 + 禁食 + 冥想 + 基础运动 + 基础感念）。
	Phase 2：完整运动 GPS + 社区地图 + 高级感念时间轴 + 订阅。
	Phase 3：AI 感悟总结、智能标签推荐、思维图谱。

###8.技术架构

####9.1.推荐技术栈

	框架：React Native + Expo（SDK 52 或 53）
	语言：TypeScript（严格模式）
	状态管理：Zustand（推荐） + Context（局部）
	数据库：WatermelonDB（推荐，适合大量感念 + 时间轴）或 AsyncStorage + MMKV
	动画：Reanimated 4 + Gesture Handler
	地图（Phase 2）：Mapbox（性价比高）或 Apple Map / Google Maps
	通知：expo-notifications
	分享：expo-sharing + react-native-view-shot（生成卡片图片）
	国际化：i18next
